# React E-Commerce Website

This project is a React-based E-Commerce application design built using modern web development technologies.

## Used Technologies

The main technologies used in this project are:

- **React.js** 
- **Vite**
- **React Router Dom** 
- **React Slick** 
- **Styled Components**
